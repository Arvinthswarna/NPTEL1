Procedure

We provide the evaluation of NPTEL course taken by the particular student in a particular semester.
So we have created 'STUDENT NPTEL DATABASE MANAGEMENT SYSTEM' to evaluate the credits digitally instead of manually.
Students can earn credit only who have added the course in this web site.
It's compulsory for all the student to register in this website.
Students are allowed to explore the courses offered by esch department.
They are allowed to take only the course specified in the list.
We will check the credentials earned throughout the academic year and add it your exemption at eight and seventh semester.
Then the student are allowed to edit their profile at the beginning of each odd semester.
If the student wants to add the course that he/she willing to exempt, he/she can add the new course via Add Course selection.
If the same course is taken for another time, it is not considered.
It will throw an error when a student approaches the same course that is completed already.
